PRESENTATION APPLICATION

Choix de jeux :
Jeu Plus ou Moins - MASTERMIND

Demarrage du jeu : Lancer Menu.java dans package JeuProjet

Deux Applications en une :

Le jeu du plus et du moins : trouver la combinaison secrete à l'aide des indications +/- données soit par l'ordinateur si c'est l'utilisateur qui doit deviner ou par l'utilisateur si c'est au tour de l'ordinateur de trouver le code secret
Mastermind : recherche d'une combinaison secrete de n chiffres grace à la reponse donnée des elements bien placés et mal placés de l'ordinateur ou de l'utilisateur.

Mode de jeu : il existe 3 modes de jeu possible : 
1er mode => Defenseur : L'ordinateur vous demande de trouver son code secret

2eme mode => Challenger : c'est à l'ordinateur de trouver le code secret de l'utilisateur

3 eme mode => Duel : l'utilisateur et l'ordinateur joue tour à tour pour trouver le code secret de l'autre, le premier qui a trouvé a gagné

Les trois modes sont réglables, en effet la taille du code secret, le nombre d'essais possibles et le chiffre maximum d'un element de la combinaison secrete peuvent être modifiés dans le fichier de configuration.
De meme, dans le fichier de configuration, il est possible de passer en mode developpeur afin d'avoir au debut du jeu la combinaison secrete.
